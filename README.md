# Portfolio Website
   - Developed a responsive portfolio website using HTML, CSS, and JavaScript, showcasing a mobile-first design approach.
    
 **Run the application:**
   ```
   git clone https://github.com/Arjunan1234/portfolio.git
   cd Django-portfolio
   click index.html
  ```
